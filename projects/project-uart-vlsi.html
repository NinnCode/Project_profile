VLSI Design for UART Circuit with Integrated FIFO, Registers and Security

Personal Project – VLSI Design & Verification
Languages: Verilog (RTL), SystemVerilog (Testbench)
Tool: Xilinx Vivado

Project Goal

The objective of this personal project was to design a modern, feature-rich UART (Universal Asynchronous Receiver-Transmitter) IP core. The design goes beyond a basic UART by incorporating essential components like FIFOs and registers, and experimentally integrating a security layer for data encryption. A key focus was placed on robust verification using a SystemVerilog testbench.

Core Architecture

The UART IP core was designed in **Verilog** with a modular architecture, comprising the following key blocks:

  • Register Block: A set of configurable registers to control UART parameters such as baud rate, parity, and stop bits.
  • TX FIFO: A First-In, First-Out buffer to store data waiting to be transmitted.
  • RX FIFO: A buffer to store received data, preventing data loss if the CPU is busy.
  • Transmitter (TX) Block: Handles parallel-to-serial data conversion, adds start/stop/parity bits, and sends data out.
  • Receiver (RX) Block: Handles serial-to-parallel data conversion and performs error checking.
  • Security Layer (Experimental): An integrated LFSR-based keystream generator for basic data encryption/decryption.

My Role and Key Tasks

As a personal project, I was responsible for the entire design and verification lifecycle:

  1. Research & Specification: Conducted in-depth research on modern UART standards and defined the functional specifications for the IP core.
  2. RTL Design in Verilog: Wrote the synthesizable RTL code for all modules of the UART using Verilog.
  3. Verification with SystemVerilog: Developed a comprehensive testbench using SystemVerilog to validate the design's functionality. This included creating constrained-random stimulus, checking for correct data transmission/reception, and verifying the security features.
  4. Simulation and Debugging: Performed all simulations and debugging within the Xilinx Vivado environment.
  5. Security Integration: Experimentally integrated a Linear-Feedback Shift Register (LFSR) to act as a pseudo-random keystream generator for data encryption and decryption.

Results & Future Work

  • The UART circuit passed all directed tests defined in the SystemVerilog testbench, including successful data transmission, reception, and the encryption/decryption process under various configurations.
  • Area for Improvement: The current verification environment relies on directed testing. The next step is to enhance the testbench to support coverage-driven and constrained-random verification methodologies to uncover more complex bugs and edge cases.
  • Security Enhancement: The LFSR-based security is a proof-of-concept. Further research into more robust cryptographic hardware accelerators (like AES) would be necessary for real-world applications.
