Line Follower Robot with Obstacle Avoidance

Competition Project – F-Race Model Car Competition
Team size: 3 | My role: Team Leader

Project Goal

The objective was to design, build, and program an autonomous robot capable of smoothly navigating a black line track and accurately detecting and stopping for obstacles in its path. This project was developed for the F-Race model car competition.

Hardware Components

The robot was constructed using the following key components:

  • Microcontroller: Arduino Uno R3 - The brain of the robot, processing all sensor data and controlling the motors.
  • Motors: V1 Geared DC Motors - Provided the necessary torque and speed for movement.
  • Line Sensor: An array of IR (Infrared) sensors to detect the black line on a white surface.
  • Obstacle Sensor: HC-SR04 Ultrasonic Sensor - To measure the distance to objects in front of the robot.
  • Motor Driver: L298N Motor Driver Module - To control the speed and direction of the DC motors.

My Contributions

  • As Team Leader, I developed the initial concept, created the project timeline, and coordinated the team's efforts.
  • I was responsible for researching and implementing the core control algorithm in Arduino C++. This involved tuning PID (Proportional-Integral-Derivative) parameters to ensure the robot followed the line smoothly and responded quickly to curves without oscillating.
  • I integrated the ultrasonic sensor data into the main loop to create a reliable obstacle avoidance system.

Result

The final robot successfully navigated the entire track, maintained stability at high speeds, and stopped consistently when an obstacle was placed in its path, fulfilling all project requirements. This project won 3rd place in the F-Race model car competition.
