Clipboard Translator & Vocabulary Saver

    Personal Project | Python Application
    Role: Individual Developer

The Problem

When reading English documents online or on a local machine, translating an unknown word often requires switching context—copying the text and pasting it into a web translator. While browser extensions can help, they don't work for local documents (like PDFs or Word files). Conversely, desktop translation software may not function online or integrate smoothly for documents on the Internet.

The Solution

This Python application bridges that gap by providing a seamless translation experience for any copyable text on your computer. It runs in the background and uses global keyboard shortcuts to instantly translate content from the clipboard, whether it's from a website, a PDF, or any other application.

How It Works

The workflow is designed for maximum efficiency:

  1. The user highlights and copies (Ctrl+C) any English text they want to translate.
  2. The user then presses a predefined keyboard shortcut to trigger an action.

Keyboard Shortcuts:

Shortcut Action
Ctrl + B Translates the clipboard content and shows a notification popup.
Ctrl + M Translates, shows the popup, and saves the English text and its translation to a .txt file for later review.

Example Usage:

1. Copy this sentence: He has versatile experience.

2. Press Ctrl + B.

3. A popup will appear on your screen showing the translation:

If you had pressed Ctrl + M instead, the same translation would also be appended to your vocabulary list file, making it easy to build a personal dictionary for future study.

Technologies Used

  • Python: Core application logic.
  • Tkinter: To create the simple, non-intrusive popup window.
  • keyboard library: To listen for global keyboard shortcuts (hotkeys) even when the app is not in focus.
  • pyperclip library: To access the content of the system clipboard.
  • googletrans: To perform the translation.
