QR Code and Multiple-Choice Answer Sheet Image Processing

Course Project – Computer Vision | VNUHCM - University of Science
Team size: 3 | My role: Team Leader

This was a final project for the Computer Vision course in the Electronics Engineering program.

Project Overview:

  • Task 1: Develop a Python-based application to enhance and decode QR codes from noisy or geometrically distorted images.
  • Task 2: Create a program to automatically process scanned multiple-choice answer sheets and return the test results of individual students.

My Contributions:

  • Took the lead in planning and task delegation within the team.
  • Conducted in-depth research on QR code decoding algorithms, noise filtering, and image correction techniques.
  • Implemented Python scripts to convert distorted or noisy QR images into correctly aligned and decodable QR codes.

Results:

  • Successfully decoded noisy QR and distorted QR codes with 99% accuracy using custom Python-based algorithms.
  • Achieved accurate grading for 100+ scanned answer sheets with minimal false detection.
  • Achieved the maximum score for the project based on technical accuracy, completeness, and presentation.
